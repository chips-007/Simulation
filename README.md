# 🏙️ [Симуляция динамики мнений](https://chips-007.github.io/Simulation/) <-- Кликай и играй!

![React](https://img.shields.io/badge/react-%2320232a.svg?style=flat&logo=react&logoColor=%2361DAFB) ![Vite](https://img.shields.io/badge/vite-%23646CFF.svg?style=flat&logo=vite&logoColor=white) ![JavaScript](https://img.shields.io/badge/javascript-%23323330.svg?style=flat&logo=javascript&logoColor=%23F7DF1E) ![License](https://img.shields.io/badge/license-MIT-green)

Браузерная симуляция распространения идей и мнений в обществе. Жители города перемещаются по улицам, общаются друг с другом, попадают под влияние идеологических лидеров, уличной рекламы и социальных связей. Наблюдайте за тем, как формируются фракции, происходят стычки и идеологические революции в режиме реального времени.

---

## 🚀 Основные возможности

* **Динамика фракций:** Непрерывная борьба "зеленого" и "красного" мнений. Включает эффект маятника: если одна сторона начинает тотально проигрывать, в обществе происходит радикализация и появляются новые лидеры сопротивления.
* **Социальные взаимодействия:** Агенты общаются 1-на-1 на улицах, лидеры "вещают" свое мнение на здания, а друзья влияют друг на друга на расстоянии (эффект социальных сетей).
* **Умная экосистема:** Дороги со временем изнашиваются (появляются ямы, блокирующие движение), а специальные агенты-ремонтники автономно патрулируют город и чинят инфраструктуру.
* **Инструменты влияния:** Игрок может устанавливать пропагандистские билборды с настраиваемым радиусом действия и спавнить наемных промоутеров.
* **Полный контроль:** Боковая панель позволяет настраивать плотность населения, количество зданий, лидеров и ремонтников на этапе генерации, а также следить за подробной аналитикой.

## 🎭 Роли агентов

| Роль | Описание поведения |
| :--- | :--- |
| **Житель** | Обычный горожанин (круг). Двигается по своим делам, заходит в здания. Мнение плавно меняется под воздействием окружения, билбордов и друзей. |
| **Лидер** | Идеологический фанатик (квадрат с рамкой). Не меняет свое мнение, вещает на толпу вокруг себя. Может вступать в дебаты с другими лидерами. |
| **Промоутер** | Наемный агитатор (ромб). Целенаправленно ходит по улицам и навязывает свое мнение прохожим при столкновении. |
| **Ремонтник** | Нейтральный рабочий (квадрат с точкой). Быстро бегает по городу, ищет сломанные участки дорог и восстанавливает их. |

## 🏗️ Объекты среды

| Объект | Влияние на симуляцию |
| :--- | :--- |
| **Здания** | Укрытия для жителей. Зашедшие внутрь агенты временно пропадают с улиц. Лидеры могут влиять на всех, кто находится с ними в одном здании. |
| **Билборды** | Статичные объекты. Излучают мнение выбранной фракции в заданном радиусе на всех проходящих мимо жителей. |
| **Ремонт дорог (Ямы)** | Препятствия. Заставляют обычных жителей разворачиваться, замедляя перемещение между районами. |

## 🎮 Управление

| Действие | Управление |
| :--- | :--- |
| **Выбор объекта** | ЛКМ (клик по билборду для изменения его цвета и радиуса) |
| **Строительство** | Двойной клик ЛКМ по карте (устанавливает выбранный в панели объект) |
| **Удаление** | ПКМ (удаляет кликнутый билборд, яму, агента или здание) |
| **Навигация по карте** | Удержание мыши (Pan) / Колесо мыши (Zoom) |
| **Управление симуляцией** | Панель слева: кнопки "Старт", "Пауза", "Генерация" |

```text

├── simulation/
│   ├── Base.js                 # Процедурная генерация карты, зданий, дорог и стартовый спавн

├── App.jsx                     # Главный интерфейс, панели генерации и аналитики
└── App.css                     # Базовые стили
```
